## SEO & GEO Strategy

### On-Page SEO
- H1 unique par page avec mots-clés principaux
- H2/H3 structurés pour profondeur sémantique
- Méta-description unique par page (<160 chars)
- URLs SEO-friendly courtes et descriptives
- Balises ALT sur toutes les images
- Liens internes contextuels entre services

### Schema.org (Données Structurées)
- [x] LocalBusiness
- [ ] Service (par page de service)
- [x] FAQPage
- [ ] Review/AggregateRating
- [ ] BreadcrumbList

### GEO (Generative Engine Optimization)
- Réponses directes dès le premier paragraphe (pyramide inversée)
- Contenu conversationnel pour recherche vocale
- Données structurées complètes
- E-E-A-T : certifications RGE, Qualibat, garantie décennale

### SEO Local
- Pages régionales robustes avec témoignages géolocalisés
- Optimisation Google Business Profile
- Données structurées LocalBusiness
